# Ahmed Murtaza - Email Signature

A professional email signature design implemented from Figma specifications.

## Features

- Modern, clean design with gradient background
- Contact information with icons
- Social media links
- Responsive design for mobile devices
- Professional typography using Inter font

## Structure

```
EmailSignature/
  index.html          # Main HTML structure
  style.css           # CSS styling and responsive design
  README.md           # This file
```

## Usage

1. Open `index.html` in a web browser to view the signature
2. Copy the HTML content for use in email clients
3. The design is optimized for both desktop and mobile viewing

## Customization

To personalize this signature:

1. Update the contact information in `index.html`
2. Replace social media links with actual URLs
3. Adjust colors in `style.css` if needed
4. Update the logo and backdrop images with your own assets

## Browser Compatibility

- Chrome/Chromium
- Safari
- Firefox
- Edge
- Mobile browsers (iOS Safari, Chrome Mobile)

## Email Client Compatibility

This signature is designed to work with most modern email clients including:
- Gmail
- Outlook
- Apple Mail
- Thunderbird
- Mobile email apps

Note: Some email clients may have limited CSS support. The signature degrades gracefully in such cases.
