# Font Installation Guide

## Required Font Files

To use the Overused Grotesk font in this email signature, you need to upload the following font files to this `fonts/` folder:

### Font Files Needed:
- `OverusedGrotesk-Regular.woff2`
- `OverusedGrotesk-Regular.woff`
- `OverusedGrotesk-Book.woff2`
- `OverusedGrotesk-Book.woff`

### Font Weights:
- **Regular (400)**: Used for the name "Ahmed Murtaza"
- **Book (300)**: Used for title, contact information, and social media text

## Installation Steps:

1. **Obtain the font files** from your design team or font license provider
2. **Upload the files** to this `fonts/` folder
3. **Verify the file names** match exactly as listed above
4. **Test the signature** by opening `index.html` in a browser

## Font Format Support:

The CSS is configured to use WOFF2 and WOFF formats for optimal browser support:
- **WOFF2**: Modern browsers, smaller file size
- **WOFF**: Fallback for older browsers

## Troubleshooting:

If the fonts don't load:
1. Check that all font files are in the correct folder
2. Verify file names match exactly (case-sensitive)
3. Ensure font files are not corrupted
4. Check browser console for font loading errors
